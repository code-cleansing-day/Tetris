# 테트리스 게임 설계해보기

## 게임 진행 관리

- 시작 부터

  - 게임오버
  - 아무키나 한 번 누르면 시작한다
  - 영역 밖으로 벗어나면 게임오버

- 점수

  - 한줄 당 몇점
  - 동시 삭제시 보너스 점수

- 시간

  - 시간 흘러감
  - 일시 정지시 정지

- 블록 쌓이는 노드

  - n x n
  - 각 노드 옵저버블
  - 필드에서 어떤 로우의 모든 노드가 채워진 경우 줄 제거

- 테트리미노
  - 7종류 구현
  - 방향키로 방향 전환 가능
  - 특정 키로 빠르게 내리기
  - 일정 시간 주기로 한 로우씩 내려감
